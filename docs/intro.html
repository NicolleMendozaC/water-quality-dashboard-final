Predicción de la Calidad del Agua mediante Machine Learning#

Descripción del proyecto#

Este proyecto presenta el desarrollo de un modelo de aprendizaje automático para predecir la calidad del agua utilizando variables fisicoquímicas.

El trabajo incluye:

  • Análisis exploratorio de datos (EDA).

  • Entrenamiento y evaluación del modelo.

  • Dashboard interactivo desarrollado con Dash.

  • Visualización de resultados.

Objetivo#

Desarrollar una herramienta que permita explorar el comportamiento de las variables de calidad del agua y realizar predicciones mediante un modelo de Machine Learning.
