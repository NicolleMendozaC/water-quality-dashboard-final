# Conclusiones

Este proyecto permitió desarrollar un flujo completo de ciencia de datos aplicado a la calidad del agua.

## Resultados obtenidos

- Se realizó un análisis exploratorio del dataset.
- Se entrenó un modelo de clasificación supervisada.
- Se evaluó el desempeño del modelo mediante métricas estándar.
- Se integró el modelo en un dashboard interactivo.

## Interpretación general

El proyecto tiene un enfoque académico y busca demostrar el flujo completo de un proyecto de Machine Learning, desde la exploración de datos hasta la implementación de una solución funcional.

## Aprendizajes

Se reforzaron habilidades en análisis de datos, modelado predictivo y visualización de resultados.
