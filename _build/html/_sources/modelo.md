# Modelo de Machine Learning

En esta sección se describe el modelo utilizado para la clasificación de la calidad del agua.

## Descripción

Se implementa un modelo de aprendizaje supervisado que utiliza variables fisicoquímicas del dataset para clasificar la calidad del agua.

Entre las variables utilizadas se encuentra el pH, junto con otros parámetros relevantes del conjunto de datos.

## Propósito

El modelo tiene un enfoque académico y demostrativo, con el objetivo de ilustrar el flujo completo de un proyecto de Machine Learning desde la exploración hasta la predicción.

## Proceso general

- Preparación de datos
- Entrenamiento del modelo
- Generación de predicciones
