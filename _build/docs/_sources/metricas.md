# Evaluación del Modelo

En esta sección se evalúa el rendimiento del modelo de clasificación.

## Métricas utilizadas

Se utilizan métricas estándar para problemas de clasificación:

- Accuracy: proporción de predicciones correctas.
- Precision: calidad de las predicciones positivas.
- Recall: capacidad del modelo para identificar correctamente cada clase.

## Interpretación

Estas métricas permiten evaluar el comportamiento del modelo frente a datos no vistos y determinar su desempeño general dentro del problema de clasificación.
